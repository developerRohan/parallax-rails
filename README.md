# parallax.js Gem

## Installation

Add this line to your application's Gemfile:

```ruby
    gem 'parallax-rails'
```

And then execute:
```
    $ bundle install
```

## Usage

Add to your app/assets/javascripts/application.js

```
    //= require parallax
    //= require jquery.parallax
```
